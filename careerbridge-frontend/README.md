# CareerBridge Frontend

Vite + React + Tailwind + GSAP frontend organized with modular components and pages.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Structure

- src/App.jsx
- src/main.jsx
- src/components
- src/pages
- src/hooks
- src/data
- src/styles
